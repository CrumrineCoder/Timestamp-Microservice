This allows you to parse in a custom URL query string and receive the UNIX and Natural time no matter which you send in.

How to Use


You input a call in the format of ("https://timestamp-microservice-crumrine.glitch.me/December%2015,%202015") or ("https://timestamp-microservice-crumrine.glitch.me/1450137600). You receive an output after loading the page that looks like: {"unix":1450137600,"natural":"December 15, 2015"}

Remix this in Glitch
